Planning the network

Switch features and configurations vary between manufacturers. Refer to the manufacturer's user manual when configuring the required features to support MoIP.

Network switch minimum requirements

  • Connect all MoIP devices to the switches making up the MoIP network topology, described below. 

  • Connect the MoIP controller to the MoIP core switch. 

  • Connect as many of the MoIP transmitters to the MoIP core switch as possible. Preferably all of them, to distribute bandwidth evenly. 

Caution: -.Do not connect a MoIP device to an unmanaged switch. 

MoIP switch requirements

  • Layer 2 or Layer 3 Managed 

  • 1 Gigabit Ethernet port throughput (minimum) 

  • Simultaneous 7.5 Watts PoE across all ports (minimum) 

  • Multicast forwarding or filtering 

  • IGMP Snooping configuration 

  • IGMP querier configuration 

  • Jumbo Frames (8000 bytes or larger) 

  • IGMP settings for router learning 

Exclusions

The below switches can only be used with MoIP if the IGMP topology is separated by a VLAN because they do not support mRouter learning or Router Auto-Learn. They are not compatible with Ryff to MoIP. 

  • Pakedge SX switches and earlier 

  • Pakedge S3 and S3L switches 

  • 100, 110, 200, and 300 series Araknis switches 

Understanding the network topology 

The network topology consists of every connected device in your local area network.  

  • Core network switch - The backbone of your local area network. This switch connects directly to your router and all other switches connect to this switch.

  • Intermediate network switch(es) – A network switch that is between two other switches. 

  • Edge network switch(es) - A network switch with a single uplink. It does not extend the network further. 

Caution: -.Unmanaged switches can only be used as edge switches because they cannot manage IGMP traffic. 

The MoIP topology is similar because it also consists of a core, intermediate, and edge switches. The core MoIP switch should connect directly to your core network switch. All MoIP devices must connect to switches in the MoIP topology.

  • Core MoIP switch - This is where your MoIP network starts. Think of it as its own topology within the larger network dedicated to IGMP.

  • Intermediate MoIP switch – A MoIP switch that is between two other switches. 

  • Edge MoIP switch – A MoIP switch with a single switch uplink. It does not extend the network further.  

Pro Tip: -_IGMP settings differ for core, intermediate, and edge switches. Be sure to make a clearly labeled connection diagram to configure each switch correctly. Start by identifying the core network switch and move out from there. 

Number of ports

Given the flexibility and scalability of Binary’s Media over IP system, plan for a switch with more Ethernet ports than needed so more MoIP devices can be added in the future.

Using an Araknis 210 Series 48-port PoE switch, the maximum MoIP system size is 46 MoIP transmitters and MoIP receivers, including a port for the MoIP controller and switch uplink.

Pro Tip: -_Use SFP adapters for switch uplinks.

When determining the size of the switch needed, count one port for each of the following devices:

  • Number of sources (MoIP transmitters)

  • Number of displays (MoIP receivers)

  • MoIP controller

  • LAN connection to the main network topology

Power

MoIP transmitters and receivers require 7.5W for power. Power-over-Ethernet (PoE) or a separate 12V power supply (sold separately) can be used. You must use the included 5V power supply for the MoIP controller.

Multiple switch deployments

Araknis switches do not currently support multiple switch setups. Multi-gig uplinks are required between extended MoIP switches. Numerous network switch manufacturers offer switches with multi-gig uplink/SFP ports to support multiple switch deployments. Check our website for the latest list of supported switches.

Switch bandwidth

Bandwidth considerations are critically important. The maximum number of MoIP transmitters a network can support is limited to the lowest bandwidth link in the topology. This is usually the link between switches.

To make bandwidth calculations easier, connect as many of the MoIP transmitters to the MoIP core switch as possible. Preferably all of them, to distribute bandwidth evenly. 

Note: -__.This applies to all MoIP switches, even if the switch only has MoIP receivers connected because they still subscribe to the streams from transmitters.

Calculating MoIP bandwidth

To calculate the required bandwidth of the MoIP system, multiply the number of MoIP transmitters by 0.85. This is the bandwidth in Gbps.
