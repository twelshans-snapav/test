Halo Remote Firmware Release Notes

Version 1.2.3

Release Date: 23-03-23

Customer Facing Changes

  • New Feature: Added the option to Disable "Room Off" Confirmation in the remote’s Device Settings.

  • Fixes to reduce the frequency and duration of Wi-Fi dropouts (Network Error Pop-Ups).

  • Improved the time it takes to reconnect after a Wi-Fi dropout.

  • Fixed a memory leak that can lead to stuck Network Error screens.

  • Fixed the Sonos driver not working correctly from Now Playing Screen.

  • The header bar where the battery and Wi-Fi signal is located was changed to black to reduce graphic layers and improve screen-to-screen transition performance.
