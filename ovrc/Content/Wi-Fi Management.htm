Wi-Fi Management

OvrC Wi-Fi Management offers complete cloud configuration for projects with multiple access points. This allows for faster setup and troubleshooting by delivering a centralized view of the entire wireless network setup. Wi-Fi Management is available on the web and mobile apps.

Compatibility

Wi-Fi configurations can only be applied to one AP series at a time. If you’re using a mix of APs, you must push the configuration to one series first, then the other.

The APs that can be configured together are:

  • Araknis 500, 700, and x10 APs

  • Araknis x20 APs

  • Pakedge WK and WX APs

  • Pakedge WA APs

Note: -__.Araknis 100 and 300 series APs are not supported.

Caution: -.Click the Import button before making any changes. Failing to do so can result in OvrC overwriting changes made in the AP’s user interface.

Getting started

You can access the Wi-Fi Management feature one of two ways:

  • By opening the OvrC web app, selecting a Customer, then clicking Wi-Fi Management in the Network Settings window.

  • By clicking the Site Settings tab, then Wi-Fi Management under Network Settings.

If this is your first time using the Wi-Fi Management feature, you’ll be greeted with an introduction message. Click Get Started.

Caution: -.Review the Importing Configurations notice before clicking Confirm.

Adding a new network

  1. Click the Import button. This ensures that OvrC does not overwrite changes made in the AP’s user interface.

  2. Use the Wireless Series drop-down to select which type of access points to configure. Wi-Fi 5 or Wi-Fi 6. You cannot configure Wi-Fi 5 and Wi-Fi 6 APs at the same time.

  3. Select Add Network to create a new network or edit the existing araknis_initial network for new installs.

Note: -__.araknis_initial can be edited or disabled but cannot be removed.

  1. Enter and select the network properties. These properties change, depending on the AP series being configured.

    Wi-Fi 5 APs allow you to configure:

    • Wireless Network Name (SSID)

    • Wireless Radios. You can select 2.4, 5GHz, or Both

    • Status (Enabled or Disabled)

    • Whether or not to Broadcast the SSID

    • Client Isolation

    • Security Mode and Password

    Wi-Fi 6 APs add the following configurable properties:

    • Fast Roaming per SSID

    • Band Steering per SSID

    • Wi-Fi 6

    • OFDM Only

  2. Select the access points to which you want to apply the Wi-Fi network and click SAVE.

Note: -__.Any pending changes in the access point’s local interface are also applied when you click the APPLY button.

Fine-tuning the wireless radios

Optimize the network by adjusting the wireless radios in each access point. To do so, click the edit icon next to the access point in the Wireless Radio Properties table.

Run a site survey before adjusting the wireless radios. Wi-Fi 5 APs have a site survey tool in the local UI. Wi-Fi 6 APs can run a site survey from OvrC. Click the table headers to sort the information.

Pro Tip: -_The BSSID field typically displays the MAC address of the radio being scanned. If the scanned radio belongs to an AP claimed in this OvrC location, the name given to the device appears instead.

Wi-Fi Management allows you to edit the following wireless radio properties:

  • Fast Roaming

  • Wireless Mode

  • Channel

  • Channel Bandwidth

  • Transmit Power

  • Enable/disable DFS channels (Wi-Fi 6 only)

  • Enable/disable the LED (Wi-Fi 6 only)

If you’re configuring a Wi-Fi 6 AP, you can also enable or disable the DFS channels and the AP’s LED.

Click Save, then Apply to push the changes to the APs.
