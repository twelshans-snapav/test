Composer

Organize Rooms and Devices

To keep a project organized in Composer, add a room for each physical room that contains a device in the project.

To add a room to a project:

  1. Click System Design.

  2. In the Items Pane click the Locations tab.

  3. Double click the desired room (i.e. Living) to add it to the project.

Place drivers in the rooms that correspond with the physical device location. Use a name that indicates the physical location of the device (i.e. Television > Living Room TV). To rename a device, right-click on the driver in the project tree or press the F2 key with the driver selected in the project tree.

Name Devices

When naming devices in Composer, get the customer’s perspective and input. Use the name that the customer uses for those devices or fixtures. When working in a vacant home or business, use names that clearly indicate which device activates when a user presses a given button or navigator icon.

To rename a device, right-click on the driver in the project tree or press the F2 key with the driver selected in the project tree.

Complete Room Endpoints

Ensure all rooms have complete and correct endpoints.

Room endpoints are the established pathway for video and audio signals, as well as the devices that control volume, temperature, security, and other functions for each room in the project. Room endpoints must be set per room for each applicable room.

To complete room endpoints:

  1. Click Connections.

  2. Click the Control/AV tab.

  3. Click the Room level (i.e. Living Room).

  4. Click the endpoint from the Control4 & Audio Video Connections list.

  5. Drag the endpoint to the correct device in the Output Devices list.
