Network configuration guides

Switch features and configurations vary between manufacturers. Refer to the manufacturer's user manual when configuring the required features to support MoIP.

You must apply the following settings to all the managed switches in the network: 

  • IGMP snooping must be enabled and set to v2. 

  • Unregistered multicast behavior set to flood and forward 

  • MRouter/Router Auto-learn disabled 

Core switches must have Querier state enabled and set to v2. 

Core and intermediate switches must have Fast leave disabled. 

Edge switches must have the following settings configured: 

  • Querier state disabled 

  • Fast leave enabled 

Read Understanding Multicast & IGMP to better understand why the network must be configured this way.

Below are configuration guides for x10 series Araknis switches and Pakedge MS series switches.

Araknis 210 / 310 Series Switch Configuration G uide 

On each switch, enable the following settings: 

  1. Navigate to Advanced > IGMP Snooping. 

  1. For Status, click Enabled. 

  1. Verify the Version is set to V2. 

  1. Set the Unregistered Multicast Behavior to Flood. 

 

  1. Under VLAN settings, enable IGMP Snooping Status on the VLAN ID the MoIP devices are configured on. 

    1. For core and intermediate switches, set Fast Leave to Disabled. 

    1. For edge switches, set Fast Leave to Enabled. 

 

  1. On core switches, set the Querier State to Enabled. Verify the Querier Version is v2. 

 

  1. Under Router Settings, at the bottom of the page, set the Router Ports Auto-Learned to Disabled on all VLANs. 

 

Pakedge MS Series Switches 

  1. Navigate to Advanced > IGMP Snooping > Configuration. 

  1. Set IGMP Snooping Global Configuration and Status to Enable, then click Apply at the top of the page.

     

  1. Move to the VLAN Status page and click the Options button to Add a VLAN entry. 

     

  1. Select the VLAN ID the MoIP devices are configured on. 

  1. Set Fast Leave to Enabledfor edge switches. 

  2. Disable Fast Leave on core and intermediate switches. 

  1. Click Add, then Apply at the top of the page. 

  1. Click the Multicast Router VLAN Configuration tab. Toggle MRouter Learning off on all ports, then click Apply. 

  1. For core switches, navigate to Advanced > IGMP Snooping Querier > Configuration. Click Enable and very the IGMP Version is set to V2. Then click Apply. 

    1. Navigate to Advanced > IGMP Snooping Querier > VLAN Configuration, then click the Options button to Add an entry. 

    2. Leave Querier Election Participation at Disabled and the Querier VLAN IP Address at 0.0.0.0. 

    3. Click Add, then click Apply at the top of the page. 

  1. On all switches, go to Advanced > Unregistered Multicast Behavior > Configuration. 

  1. Set Unregistered Multicast Behavior Configuration to Forward, then click Apply.

     

  2. If you see an entry for EXC_default_list under the Exceptions list, continue to the following steps. If not, your configuration is complete. 
     

  3. Click the Interface Configuration tab, then click Options, and Edit. 

     

  1. Select each entry for EXC_default_list. If that’s each entry in the list, click the box next to Interface to select all. Then click Edit Selected. 

  2. Click the “x” in the EXC_default_list object to delete it. 

     

  3. Click Save, then Apply at the top of the page.
