Configuring Network Controls

Network Controls allow you or the OvrC Connect user to create Wireless Device Profiles. You can assign devices that an OvrC Pro device has scanned to a profile and set network connectivity schedules for when they can connect, or manually pause their network connectivity. Use cases include profiles for a customer's children or giving a customer the ability to disrupt network connectivity to an indoor third-party wireless camera.

Note: -__.These settings only work for wireless devices. Wired or cellular connections can still be made.

Creating a Wireless Device Profile

  1. Select the OvrC Customer and Location.

  2. Navigate to Client Services > Network Controls > Add Profile.

  3. Enter a Profile Name. This is typically the name of someone in the household.

  4. The Associated Devices table shows all the client devices that have been scanned by OvrC Pro. Select the devices that belong to the individual you named the Profile for.

  5. If you'd like to set up a Schedule, follow the below instructions. If not, click Save and go to the next section.

Schedules

Schedules are commonly used to block out time for homework, sleep, or dinner. Or to schedule when indoor wireless cameras should connect to the network.

Note: -__.This only disconnects devices from Wi-Fi. Wired or cellular connections can still be made.

To configure a Schedule:

  1. Click the Add Schedule button.

  2. Enter a Schedule Name and make sure the correct Time Zone is selected.

  3. Weekdays and Weekends are configured separately. Each has its own toggle button to enable or disable these time periods.

  4. Select the individual Days you'd like the schedule applied to.

  5. Enter a Pause and Resume Time. You can type the time or click the clock icon to select the time with your cursor, then click Save.

Pausing Profiles

Profiles can be manually paused in OvrC and in the OvrC Connect app.

Network Settings

Also configurable on the Site Settings tab. Click Content Filtering & Network Security Settings to select the network security and what content to filter out on the entire network. Complete descriptions of each level are detailed on the web page.

Toggle Enable in OvrC Connect on if the customer wants control over these settings

Note: -__.Enabling this feature changes the DNS servers of the router.

Caution: -.These settings are global and affect every user on the network.
